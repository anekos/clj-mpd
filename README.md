# Silence is your alarm.

## Usage

### Initialize

Load MPD music library to cache.

```
$ mpd-timer update
```

### Set the timer

Set the timer for 2 hours 30 minutes.
(= Setup MPD playlist and play it)

```
$ mpd-timer set 2h30m
```
